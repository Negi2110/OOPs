// ANSWER:
// A thread is the smallest unit of execution within a process. Every C++ application starts with a 
// default thread—the main() function—and additional threads can be created to perform tasks in parallel.
// Threads help achieve concurrency and can significantly improve performance, especially when tasks are

# ✅ **5 Ways to Create Threads in C++11 (With Examples)**

---

## **1. Using Function Pointer**

```cpp
#include <iostream>
#include <thread>
using namespace std;

void func() {
    cout << "Thread using function pointer\n";
}

int main() {
    thread t(func);
    t.join();
}
```

---

## **2. Using Lambda Function**

```cpp
#include <iostream>
#include <thread>
using namespace std;

int main() {
    thread t([] {
        cout << "Thread using lambda function\n";
    });
    t.join();
}
```

---

## **3. Using Functor (Function Object)**

```cpp
#include <iostream>
#include <thread>
using namespace std;

class Functor {
public:
    void operator()() {
        cout << "Thread using functor\n";
    }
};

int main() {
    Functor f;
    thread t(f);
    t.join();
}
```

---

## **4. Using Member Function**

⚠ Requires passing the object pointer.

```cpp
#include <iostream>
#include <thread>
using namespace std;

class Worker {
public:
    void run() {
        cout << "Thread using member function\n";
    }
};

int main() {
    Worker w;
    thread t(&Worker::run, &w);
    t.join();
}
```

---

## **5. Using Static Member Function**

✔ Works directly because no `this` pointer is required.

```cpp
#include <iostream>
#include <thread>
using namespace std;

class Worker {
public:
    static void run() {
        cout << "Thread using static member function\n";
    }
};

int main() {
    thread t(&Worker::run);
    t.join();
}
```
